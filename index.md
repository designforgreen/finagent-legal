# FinAgent

**面向中文财经用户的信息聚合阅读器**

FinAgent 把你关注的**微信公众号、雪球、微博**账号的更新汇总成每日简报,方便集中阅读。
配合 Chrome 扩展,你的**登录身份**由自己管理,FinAgent 以你本人的身份抓取你关注的内容 —— BYOK (Bring Your Own Key) 模式。

## 产品

- **FinAgent Lite** 主服务:<https://getfinagent.com>(内测中)
- **Chrome 扩展**:用于同步三平台登录 cookies(Chrome Web Store 审核中)

## 法律文书

- [**隐私政策**](./privacy-policy) — 数据收集、使用和保护规则

## 设计理念

1. **BYOK** — 抓取凭证由用户自己提供,不共用系统账号
2. **单一用途** — 扩展只读三个特定平台 cookies,不涉及其他数据
3. **端到端加密存储** — AES-256-GCM 加密凭证,独立密钥管理
4. **用户完全控制** — 随时解绑、导出、删除
